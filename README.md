Teledock_Web_Application
Описание
Небходимо создать веб-приложение для добавления и просмотра клиентов (редактирование по желанию). Клиентами выступают юридические лица
(ЮЛ) и индивидуальные предприниматели (ИП). ЮЛ могут иметь некоторое количество учредителей.

Необходимые поля
Клиент
- ИНН
- Наименование
- Тип (ЮЛ / ИП)
- Дата добавления (заполняется автоматически)
- Дата обновления (заполняется автоматически)
Учредитель
- ИНН
- ФИО
- Дата добавления (заполняется автоматически)
- Дата обновления (заполняется автоматически)
С типами данных для полей, связями, ограничениями и валидацией необходимо определиться самостоятельно.

Рекомендации
Для разработки использовать платформу ASP.NET Core MVC .
В качестве СУБД использовать на выбор:
SQL Express LocalDB (коробочное решение в Visual Studio )
MS SQL (на базе docker )
PostgreSQL (на базе docker )
В качестве ORM использовать Entity Framework Core и подход Code First .
Для оформления разметки использовать Bootstrap или собственные стили по желанию.
Не использовать встроенные механизмы генерации шаблонных контроллеров и представлений в Visual Studio , т.к. нам необходим ваш
личный код для обсуждения и ревью.
Загрузить исходный код на любую Git платформу с уникальным названием проекта (либо же скрыть из публичного поиска).
Если проект будет нестандартным (иметь нюансы по запуску или что-то подобное), необходимо приложить файл с описанием
